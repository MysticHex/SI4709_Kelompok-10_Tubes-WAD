## Deployment

To deploy this project run

Install composer Depencies
```bash
  composer install
```
Initalize NPM
```bash
  npm init
```
Install node dependcies
```bash
  npm install
```
Build NPM Depencies
```bash
  npm run build
```
Run NPM dev
```bash
  npm run dev
```
Run Laravel project
```bash
  php artisan serve
```
